
Spring Web MVC 는 가장 인기있는 프레임워크 중 하나이다.
MSA 에서 각 컴포넌트 (MicroService)는 자신이 관리하는 데이터가 독립적으로 존재하며 클라이언트의 요청에 따라 이를 통합하여 제공해야 하는 경우가 많다. 분리된 데이터를 통합하는 방식에는 API (RESTAPI, gRPC, AVRO) 혹은 메시지 큐 방식이 일반적으로 사용된다.
